# Falling-sand-game
by Ayoub Agouzoul &amp; Rachid Tazi

## Compilation
All commands should be run from project root.


```bash
# one-liner
scons platform=linux target=template_debug -j$(nproc) && godot . &
```

To compile: scons platform=linux target=template_debug -j$(nproc)

To run directly the game : godot4 .

To open the editor : godot4 --editor

## Understand Godot structure

- .godot/
folder generated by Godot

- godot-cpp/
official library that allow us to connect C++ to Godot via GDExtension
Godot <-> godot-cpp <-> code C++

- src/ 
folder containing code C++

- project.godot 
main file
containing the name, principal scene...

- main.tscn
main Godot scene

- SConstruct
compilation file for SCons
(equivalent Makefile)

- .gdextension 
to load c++

- .gd 
script GDScript 

- .glsl
shader glsl
compute shader
